# FEATURE_INVENTORY

## Core features
- Workspace and user accounts.
- Case creation and editing.
- Case status lifecycle.
- Case ownership and collaboration.
- Notes, tasks, files, and timeline.
- AI summary and suggested next steps.
- Search, filters, and dashboard.

## Supporting features
- Tags and priority/risk levels.
- Case templates.
- Activity feed.
- Exportable summaries.
- Basic notifications/reminders.

## Admin/operator features
- User and role management.
- Workspace settings.
- Case template management.
- Audit/event log.
- Data export.
- Archive/restore controls.

## Analytics/reporting needs
- Open cases by status.
- Overdue tasks.
- Cases by owner.
- High-risk cases.
- Average case age.
- Recently inactive cases.

## Notifications/messaging needs
- Task due soon.
- Task overdue.
- User mentioned or assigned.
- Case status changed.
- AI summary generated and ready for review.

## Search/filtering needs
- Search by title, client/entity, case ID, notes, tags.
- Filter by owner, status, priority/risk, due date, updated date.

## Security/privacy-sensitive features
- Role-based access control.
- Audit trail.
- Secure file storage.
- Data retention settings.
- AI provider/data handling configuration.
- Clear separation between user-entered facts and AI-generated text.

## Future features
- Email/calendar integrations.
- Document extraction and RAG.
- Workflow automation.
- Billing.
- Client portal.
- Advanced permission groups.
