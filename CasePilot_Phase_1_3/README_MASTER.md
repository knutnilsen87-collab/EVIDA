# CasePilot Phase 1-3

## Status
This folder contains the first structured product foundation for CasePilot, based on the intake-pack ZIP.

## Working assumption
CasePilot is treated as an AI-assisted SaaS/workspace for handling cases, client matters, investigations, support cases, or structured operational workflows.

This is an assumption, not a final business decision.
