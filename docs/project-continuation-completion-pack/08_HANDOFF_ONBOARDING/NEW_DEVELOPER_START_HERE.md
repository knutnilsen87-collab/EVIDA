# New Developer Start Here

## First hour
1. Read `README.md`.
2. Read `ARCHITECTURE.md` and `SECURITY.md`.
3. Read all ADRs in `DECISIONS/`.
4. Run `ops\Verify-ProductionBoundary.ps1`.
5. Run `npm.cmd test` and `npm.cmd run build` in `evida-core/desktop-tauri`.

## First day
- Open `src/App.tsx`, `src/components/CaseRoomView.tsx`, `src/components/Sidebar.tsx`, `src/styles.css`.
- Trace import/Saksrom/readiness flow.
- Read `docs/SAKSROM_CHAT_FIRST_HANDOFF.md`.
- Review release checklist.

## First PR rule
Make one narrow change, preserve pre-alpha/security language, and verify with the active commands.
