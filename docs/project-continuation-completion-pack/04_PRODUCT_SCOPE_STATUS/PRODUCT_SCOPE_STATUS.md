# Product Scope Status

## Current product scope
A local-first desktop legal workspace for evaluation:
- import documents,
- create/source local case basis,
- ask questions in Saksrom,
- show uncertainty and missing basis,
- route work into legal workrooms,
- package local Windows builds for evaluation.

## Explicit non-production status
The app is not ready for real client data or production legal work.

## MVP focus
The MVP should prove a trusted local legal briefing loop:
1. Import approved test documents.
2. Produce traceable source basis.
3. Show readable case summary.
4. Answer with source references and uncertainty.
5. Let user take next legal workroom actions.
6. Preserve local/security boundary.

## Avoid scope expansion
Do not add broad SaaS/admin/product-growth features until security, storage, backend ownership and release gates are hard enough.
