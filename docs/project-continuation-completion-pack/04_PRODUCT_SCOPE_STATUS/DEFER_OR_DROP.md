# Defer Or Drop

## Defer until production-hardening phase
- Multi-tenant SaaS administration.
- Billing/license enforcement beyond documented control-plane direction.
- External AI provider workflows.
- Broad integrations with third-party document stores.
- Public release signing and distribution automation.

## Drop or quarantine unless revalidated
- Any old architecture implying FastAPI owns enterprise/control-plane policy.
- Archive/legacy generated source folders as active implementation material.
- Demo-only claims that suggest production security is complete.

## Keep but label as evaluation-only
- Local guided intro/login.
- Local test data import.
- Local AI-like answer scaffolding/safe mode.
